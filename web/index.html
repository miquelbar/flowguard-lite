Router & SIEM Integrations

Guides and configuration templates to connect your network devices

Ubiquiti UniFi Setup

Depending on your UniFi Gateway model, choose the appropriate integration path:

Path A: SIEM / Activity Logging (Recommended for Cloud Gateways)
  1. Navigate to Settings > CyberSecure > Traffic Logging in the UniFi Controller.
  2. Enable Activity Logging (Syslog).
  3. Set the SIEM Server IP to [FLOWGUARD_IP].
  4. Set the Port to 514 (mapped to FlowGuard's 5514 container port).
  5. Enable categories like Critical, Devices, and Security Detections.
Path B: NetFlow / IPFIX Exporter
  1. Go to Settings > Networks > Global Network Settings.
  2. Find the NetFlow or Flow Export section.
  3. Set the collector target to [FLOWGUARD_IP] and port 2055.

MikroTik RouterOS Setup

Run the following commands in the RouterOS terminal to enable Traffic-Flow (NetFlow v9):

/ip traffic-flow target
add address=[FLOWGUARD_IP] port=2055 version=9 v9-template-timeout=1m v9-template-entries=20

/ip traffic-flow
set enabled=yes

Replace [FLOWGUARD_IP] with the IP address of the machine running FlowGuard Lite.

OPNsense & pfSense NetFlow Setup

OPNsense Steps
  1. Navigate to Reporting > NetFlow.
  2. In the Listening Interfaces list, select your LAN/WAN interfaces.
  3. In the WAN Interfaces list, select your active WAN interface.
  4. Set the Capture Target to [FLOWGUARD_IP]:2055.
  5. Click Apply to start the NetFlow service.
pfSense Steps
  1. Install the softflowd package via the Package Manager.
  2. Navigate to Services > Softflowd.
  3. Select the interfaces to monitor (e.g. LAN).
  4. Set the Host/Port to [FLOWGUARD_IP] and port 2055.
  5. Set the NetFlow Version to 9.

Suricata IDS Integration

Configure Suricata's suricata.yaml configuration file to export the eve.json log file:

outputs:
  - eve-log:
      enabled: yes
      filetype: regular
      filename: eve.json
      types:
        - alert:
            payload: no
            packet: no
            metadata: yes

Once configured, input the path to the eve.json file inside FlowGuard's Collectors Setup settings to correlate IDS alerts with network traffic.
